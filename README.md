# 多地点出行顺序决策工具

这是一个线上使用的路线顺序决策工具，只比较路线顺序、距离、时间和用户手动录入的实际价格。

## 核心能力

- 输入一个固定出发点和多个待访问地点。
- 调用高德 POI 搜索，确保地点是真实坐标。
- 为不同访问顺序生成路线候选。
- 调用高德驾车路线 API，计算真实道路距离、预计时间和轨迹。
- 默认按时间优先推荐，距离作为第二排序条件。
- 用户可以手动录入 App 实际价格，作为可选对比依据。
- 支持历史记录和分享链接。

## 产品边界

| 做什么 | 不做什么 |
|---|---|
| 多地点访问顺序比较 | 自动预测滴滴整单价格 |
| 高德真实路线距离和时间 | 分段价格相加 |
| 用户手动录入 App 实际价格 | 虚构平台价格 |
| MCP 能力检测 | 把 MCP 当作 itinerary pricing |
