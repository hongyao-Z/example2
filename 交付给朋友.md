# 给朋友看的访问方式

当前可访问链接：

```text
https://mysimon-python-pavilion-destination.trycloudflare.com
```

把上面的链接发给朋友即可。

## 使用条件

- 你的电脑需要保持开机。
- 当前项目服务窗口需要保持运行。
- 当前 Cloudflare Tunnel 需要保持运行。
- 这个链接是临时公网链接，适合马上给朋友试看。

## 朋友怎么用

1. 打开链接。
2. 输入城市，例如“北京市”。
3. 搜索并选择出发点。
4. 搜索并选择多个待访问地点。
5. 点击“生成访问顺序”。
6. 查看不同顺序的距离、时间和路线。
7. 如需比较实际 App 价格，可以手动录入价格。
